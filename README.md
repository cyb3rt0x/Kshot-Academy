# KSHOT Academy

Static website for the KSHOT Academy. Ready to publish on GitHub Pages.

## How to publish

1. Create a GitHub repository.
2. Upload all files from this folder.
3. In GitHub, open **Settings → Pages**.
4. Select **Deploy from branch**.
5. Choose the `main` branch and `/root` folder.
6. Save.

## Structure

- `index.html` — main HTML file.
- `styles.css` — Kingshot-inspired visual style.
- `data.js` — all academy, lesson, and roadmap content.
- `app.js` — navigation, lesson rendering, search, progress.

## Editing content

Edit `data.js`. Add or update lessons inside the `LESSONS` object and academy structure inside `ACADEMIES`.
