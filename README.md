# KSHOT Academy V3

Static GitHub Pages site for the KSHOT Academy.

## What changed in V3

- Visual system closer to the Kingshot UI references: parchment background, wooden panels, bright turquoise buttons, gold/green/gray status colors, stronger outlines, and game-like depth.
- Mobile-first layout with a bottom navigation bar.
- Custom SVG icon set inspired by the Kingshot visual language, without copying game assets.
- Academy King mascot illustration for the homepage.
- Global search now searches lessons and FAQ entries.
- Knowledge Base / FAQ section added.
- Language selector added for 7 global languages. English is the default and fallback language.
- Full Master Index remains available with Published and In Development lesson states.

## How to publish on GitHub Pages

1. Upload all files to the repository root.
2. Go to Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: / (root).
6. Save.

## Notes

The content is intentionally system-based instead of fixed “use Hero X” recommendations. This keeps the Academy useful across future hero generations, updates, and meta changes.
